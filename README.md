# backendAdmin

![screenshot](https://tantails.github.io/backendAdmin/screenshot.png "screenshot")

## 功能
* 前後台網頁檔案及權限管理
* 使用者及權限及管理

## 安裝
* PHP版本需求為7.0以上
* 將檔案放入網頁根目錄後，開啟`include/base.php`修改 MySQL 的帳號密碼
